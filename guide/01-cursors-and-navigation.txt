EVM Guide 01: Creating Cursors and Navigation
===============================================

Open this file in Emacs with evil-mode and evm loaded.
Practice on the text between ~~~~ lines. Esc exits evm.


Lesson 1: Find Word (C-n) and Adding Matches
----------------------------------------------

C-n on a word enters evm and selects it. Other occurrences
are shown as dim previews. Press C-n (or n) to add the next
match; N adds the previous one.

> Place cursor on "nval" below and press C-n, then \ A to
  select all. Press c, type "total", press Esc twice.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  nval = 0
  for item in items:
      nval += 1
      if nval > limit:
          break
  print("result:", nval)
  reset(nval)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected: every "nval" becomes "total".

You can also start from a visual selection: select "get"
with v, then C-n matches partial words too.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  budget = get_total()
  target = widget.get_value()
  gadget_list = get_gadgets()
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 2: Skip (q), Remove (Q), Select All (\ A)
--------------------------------------------------

q removes the leader cursor and jumps to the next match.
Q removes the leader cursor (no jump). \ A selects all
remaining occurrences at once.

> Place cursor on "port" line 1 below, press C-n.
  You want lines 1, 3, 5 but NOT 2, 4.
  Sequence: C-n, q, C-n, q. Result: 3 cursors.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  port = 8080
  port = 3000
  port = 8080
  port = 5432
  port = 8080
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

> Now try \ A: place cursor on "id" below, press C-n,
  then \ A. All 8 occurrences selected instantly.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  id: 1001
  id: 1002
  id: 1003
  id: 1004
  id: 1005
  id: 1006
  id: 1007
  id: 1008
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 3: Vertical Cursors (C-Down / C-Up)
--------------------------------------------

C-Down adds a cursor one line below at the same column.
C-Up goes upward. Column is preserved even through short
lines (cursor lands at EOL on short lines, but remembers
the target column).

> Place cursor on the first character of "alpha" below,
  press C-Down four times. 5 cursors in a column.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  alpha
  bravo
  charlie
  delta
  echo
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

> Place cursor on "1" of "10" below, press C-Down four
  times. Cursors land on the first digit of each number.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  width   = 10
  height  = 20
  depth   = 30
  margin  = 40
  padding = 50
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 4: Cursors from Visual Selection (\ c)
-----------------------------------------------

Select a region in visual mode, then \ c creates cursors:

  V     (visual-line)  -- cursor at first non-blank of each line
  C-v   (visual-block) -- cursor at block column on each line
  v     (visual-char)  -- cursor at start and end of selection

> V-select lines 2-5 below (place cursor on "bravo", press
  V, then 3j), then press \ c. 4 cursors, one per line.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  alpha
  bravo
  charlie
  delta
  echo
  foxtrot
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

> C-v select: place cursor on "1" of "10", press C-v,
  then 4j, then \ c. 5 cursors at the number column.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  width   = 10
  height  = 20
  depth   = 30
  margin  = 40
  padding = 50
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 5: Cursor Mode vs Extend Mode
--------------------------------------

EVM has two modes (shown in the mode-line as C or E):

  Cursor mode (C) -- point cursors. d/c/y need a motion.
  Extend mode (E) -- selections. d/c/y act directly.

C-n enters extend mode (words are selected).
C-Down enters cursor mode (points only).
Tab switches between them.

Cursor -> extend: each point becomes a 1-char selection.
Extend -> cursor: selections collapse to points.

> C-n on "slot" below, add 3 matches. You're in extend mode.
  Press Tab -- selections collapse (cursor mode).
  Press Tab again -- 1-char selections (extend mode).

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  slot = "Alice"
  print(slot)
  slot = "Bob"
  print(slot)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

> Extend mode d/c/y (no motion needed):
  C-n on "old" below, press n once. Press d -- both deleted.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  value = old + old + new
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

> Cursor mode d/c/y (motion required):
  Place cursor on "1" of "10", press C-Down twice.
  Press d$ -- deletes from cursor to end of each line.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  width  = 10
  height = 20
  depth  = 30
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 6: Navigation Between Cursors
--------------------------------------

] moves the leader to the next cursor.
[ moves the leader to the previous cursor.

The leader is shown in a different color. It's where your
real cursor sits; other cursors are "followers."

> C-n on "item" below, press \ A. Use ] and [ to cycle
  through the cursors. Watch the mode-line counter change.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  item = 1
  item = 2
  item = 3
  item = 4
  item = 5
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 7: Motions Move All Cursors
------------------------------------

Every motion (h/j/k/l, w/b/e, 0/^/$, f/t/F/T, gg/G)
moves ALL cursors simultaneously. In extend mode, motions
grow or shrink selections instead.

o flips the active end of a selection (like evil visual o).

> Place cursor at column 0 line 1 below, C-Down twice.
  Press f= -- all cursors jump to "=".
  Press l l -- cursors on the first digit of each value.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  nick: Aspen
  rank: elite
  clan: Forge
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

> Extend mode motions -- C-n on first char of line 1 below,
  C-Down twice. Press Tab (extend), then e -- selection
  covers the first word. Press $ -- selection to end of line.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  addr = 0.0.0.0
  proto = tcp
  tier = staging
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 8: Putting It All Together
-----------------------------------

Add "export " prefix and ";" suffix to each line:

> 1. Place cursor anywhere on line 1 below
  2. C-Down three times (4 cursors)
  3. I -- insert at line start. Type "export "
  4. Esc (stay in evm)
  5. A -- append at line end. Type ";"
  6. Esc, Esc

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  const alpha = 1
  const bravo = 2
  const charlie = 3
  const delta = 4
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected:
  export const alpha = 1;
  export const bravo = 2;
  export const charlie = 3;
  export const delta = 4;

Rename and wrap values:

> 1. C-n on "srv" below, then n twice (3 matches, extend mode)
  2. c, type "server", Esc (cursor mode, still in evm)
  3. f= l l -- cursors on the values
  4. Tab, $ -- extend mode, selections to end of line
  5. S " -- surround with quotes (requires evil-surround)
  6. Esc

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  srv = alpha
  srv = bravo
  srv = charlie
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected:
  server = "alpha"
  server = "bravo"
  server = "charlie"
