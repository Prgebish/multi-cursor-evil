EVM Tutorial 15: Run at Cursors
=================================

These commands execute a normal command, macro, or ex
command at every cursor position. Useful for complex
operations that go beyond standard cursor mode replication.

  \ z  = run normal command at all cursors
  \ @  = run macro (from register) at all cursors
  \ :  = run ex command at all cursors

The leader key (\ by default) is set by evm-leader-key.


Lesson 1: Run Normal Command (\ z)
-------------------------------------

\ z prompts for a key sequence, then executes it as a
normal mode command at each cursor position.

> Try it: uppercase the word at each cursor.
  1. Place cursor on "a" of "alice" on line 1 below
  2. Press C-Down twice (3 cursors)
  3. Press \ z — prompt: "Normal command: "
  4. Type gUiw and press Enter — words uppercased
  5. Press Esc when done (u to undo)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  name = alice
  name = bob
  name = carol
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
  name = ALICE
  name = BOB
  name = CAROL

  The string "gUiw" runs as evil keystrokes at each cursor:
  gU (uppercase operator) + iw (inner word text object).


Lesson 2: Run Macro (\ @)
----------------------------

\ @ prompts for a register character, then replays the
macro from that register at every cursor position.

Record the macro first, then use \ @ to replay it.

> Try it: add semicolons using a recorded macro.
  1. Place cursor on "x" on line 1 below
  2. Record a macro: press qa, then A;, Esc, q
     (appends ";" to line 1)
  3. Press u to undo (line restored, macro stays in reg a)
  4. Press C-Down twice (3 cursors, cursor position doesn't matter)
  5. Press \ @ then a — macro replayed at all cursors
  6. Press Esc when done (u to undo)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  x = 1
  y = 2
  z = 3
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
  x = 1;
  y = 2;
  z = 3;

  The macro in register a (A ; Esc) runs at each cursor:
  A (append at end of line), ; (insert), Esc (exit insert).


Lesson 3: Run Ex Command (\ :)
---------------------------------

\ : prompts for an ex command, then runs it at each
cursor's line. Useful for substitutions and other
line-oriented commands.

> Try it: rename a keyword with substitution.
  1. Place cursor on "v" of "var" on line 1 below
  2. Press C-Down twice (3 cursors)
  3. Press \ : — prompt: ": "
  4. Type s/var/let/ and press Enter — replaced on each line
  5. Press Esc when done (u to undo)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  var x = 1
  var y = 2
  var z = 3
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
  let x = 1
  let y = 2
  let z = 3

  The ex command s/var/let/ runs on each cursor's line,
  replacing the first match of "var" with "let".


Lesson 4: Selective Lines with \ :
-------------------------------------

\ : only affects lines where cursors are placed. Use C-n
to place cursors on specific lines, then run the command —
other lines stay untouched.

> Try it: rename "calc" only on the lines that have it.
  1. Place cursor on "calc" on line 1 below, press C-n
  2. Press n twice (3 matches of "calc")
  3. Press Tab to switch to cursor mode
  4. Press \ : — type s/calc/compute/ and press Enter
  5. Press Esc when done (u to undo)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  result = calc(1)
  other = 42
  result = calc(2)
  label = ok
  result = calc(3)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
  result = compute(1)
  other = 42
  result = compute(2)
  label = ok
  result = compute(3)

  Lines 2 and 4 have no cursors — they stay unchanged.
  Compare with :%s/calc/compute/g which would change
  every line in the buffer.

  Quick reference:
    \ z             run normal command at all cursors
    \ @             run macro from register at all cursors
    \ :             run ex command at all cursor lines
    All three prompt for input before executing.


That's it for Tutorial 15!
