EVM Tutorial 04: Cursor Mode and Extend Mode
===============================================

EVM has two modes of operation:
  Cursor mode (C) — multiple point cursors, for typing and motions
  Extend mode (E) — multiple selections, for operating on selected text

The current mode is shown in the mode-line (the status bar at the
bottom of each Emacs window):
  EVM[C 1/3] = cursor mode, cursor 1 of 3
  EVM[E 1/3] = extend mode, region 1 of 3

If your mode-line is already crowded, you can disable this:
  (setq evm-show-mode-line nil)

Tab switches between them.


Lesson 1: How You Enter Each Mode
-----------------------------------

When you create cursors with C-n (find word), evm starts in extend
mode — each word is selected (highlighted). The leader selection
is shown in a different color to stand out.

When you create cursors with C-Down/C-Up, evm starts in cursor
mode — each cursor is a single point. The leader cursor has
a distinct color.

> Try it: place cursor on "data" below and press C-n.
  Look at the mode-line — it says EVM[E ...]. Each "data" is
  selected (highlighted). Press Esc when done.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  data = load("file")
  print(data)
  data = transform(data)
  save(data)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

> Now try: place cursor at the start of "alpha" and press C-Down
  three times. Look at the mode-line — it says EVM[C ...].
  Each cursor is a single highlighted point. Press Esc when done.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  alpha = 1
  bravo = 2
  charlie = 3
  delta = 4
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 2: Switching with Tab
------------------------------

Tab toggles between cursor mode and extend mode.

From extend to cursor: selections collapse to points. The cursor
lands at the beginning of each selection.

From cursor to extend: each point expands to a one-character
selection. You can then grow it with movement keys (w, e, $, ...).

> Try it:
  1. Place cursor on "slot" below and press C-n three times
     (you should have 3 selections in extend mode)
  2. Press Tab — selections collapse to point cursors (EVM[C])
  3. Press Tab again — each cursor becomes a one-character
     selection (EVM[E]). The original selection is not restored —
     use motions (w, e, $) to grow it again.
  4. Press Esc when done

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  slot = "Alice"
  print(slot)
  slot = "Bob"
  print(slot)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 3: Extend Mode — Delete (d)
-------------------------------------

In extend mode, d deletes all selected text at once. No motion
needed — d acts directly on the selections. After deletion, evm
switches to cursor mode automatically.

> Try it:
  1. Place cursor on "old" below and press C-n twice (2 selections)
  2. Press d — both "old" are deleted
  3. Notice the mode switched to cursor (EVM[C])
  4. Press Esc when done (then u to undo the edit)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  value = old + old + new
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 4: Extend Mode — Change (c)
-------------------------------------

In extend mode, c deletes all selections and enters insert mode.
Whatever you type replaces all selections simultaneously. This is
the most common way to rename things.

> Try it:
  1. Place cursor on "tval" below and press C-n
  2. Press \ A to select all occurrences
  3. Press c — selections are deleted, you're in insert mode
  4. Type "result"
  5. Press Esc, then Esc again

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  tval = calculate()
  if tval > 0:
      save(tval)
  print(tval)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 5: Extend Mode — Yank and Paste (y, p)
------------------------------------------------

In extend mode, y copies each selection separately. When you then
paste with p onto the same number of cursors, values are distributed
1-to-1: cursor 1 gets value 1, cursor 2 gets value 2, etc.

This is how you move values from one place to another.

> Try it: copy the names from the left column into the right.
  1. Place cursor on the "a" of "alpha" on line 1
  2. Press C-Down twice (3 cursors in cursor mode)
  3. Press Tab, then press e — selections cover "alpha", "bravo", "charlie"
  4. Press y — "Yanked 3 regions"
  5. Press Esc to exit evm
  6. Place cursor on the 4-letter placeholder after "=" on line 1
  7. Press C-n, then \ A (selects all 3 placeholders)
  8. Press p — each placeholder is replaced with its corresponding name
  Expected result: alice = alpha, bob = bravo, carol = charlie

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  alpha  =>  alice = xxxx
  bravo  =>  bob = xxxx
  charlie  =>  carol = xxxx
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 6: Growing Selections with Motions
--------------------------------------------

In extend mode, movement keys (w, e, $, l, h, ...) expand or
shrink your selections. Each selection has an anchor (fixed end)
and an active end (the end that moves).

When you enter extend mode via Tab from cursor mode, the anchor
is where the cursor was. Motions move the active end away from
the anchor, growing the selection.

> Try it:
  1. Place cursor on the "v" of "val" on line 1 below
  2. Press C-Down three times (4 cursors in cursor mode)
  3. Press Tab to switch to extend mode — each cursor selects
     one character ("v")
  4. Press e — selection grows to end of word ("val")
  5. Press e again — selection grows to end of next word
  6. Press Esc when done

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  val = 10
  val = 20
  val = 30
  val = 40
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 7: Flip Direction (o)
-------------------------------

In extend mode, o flips which end of the selection is active.
This lets you grow the selection in the opposite direction.

Like evil visual mode: o swaps cursor and anchor.

> Try it:
  1. Place cursor on "tag" below and press C-n twice
     (2 selections, each covering the word)
  2. Press l twice — selections grow right to include " ="
  3. Press o — the active end flips to the left side
  4. Press h three times — selections grow left
  5. Press Esc when done

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  my tag = "Alice"
  my tag = "Bob"
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 8: Cursor Mode — Operators Need Motions
-------------------------------------------------

In cursor mode, d/c/y are operators — they wait for a motion.
This is just like regular evil:
  dw = delete word, ciw = change inner word, y$ = yank to EOL.

In extend mode, d/c/y act immediately on selections.

This is the key difference between the two modes.

> Try it (cursor mode operators):
  1. Place cursor on the "1" of "10" on line 1 below
  2. Press C-Down three times (4 cursors)
  3. You're in cursor mode — press d$ to delete from cursor to EOL
  4. All four values are deleted
  5. Press Esc (then u to undo)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  width  = 10
  height = 20
  depth  = 30
  margin = 40
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

> Try it (extend mode — no motion needed):
  1. Place cursor on "aa" on line 1 below, press C-n
  2. Press \ A to select all occurrences
  3. You're in extend mode — press d to delete all selections
  4. Press Esc (then u to undo)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  width  = aa
  height = aa
  depth  = aa
  margin = aa
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 9: Practical Workflow
------------------------------

The typical workflow combines both modes:

  C-n (extend) → Tab (cursor) → edit → Tab (extend) → operate

Or more commonly:
  C-n → \ A → c → type replacement → Esc

> Try it: rename the variable and wrap each value in quotes.
  1. Place cursor on "srv" on line 1 below and press C-n twice
     (3 occurrences selected in extend mode)
  2. Press c, type "server", press Esc
     (still in evm, now in cursor mode)
  3. Press f= to jump all cursors to "="
  4. Press l twice to land on the value
  5. Press Tab to enter extend mode
  6. Press $ to extend selections to end of line
  7. Press S then " to surround each value with quotes
     (requires evil-surround; see Tutorial 13)
  8. Press Esc to exit evm

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  srv = alpha
  srv = bravo
  srv = charlie
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
