EVM Tutorial 01: Find Word, Add Next, Select All
==================================================

Welcome to the EVM interactive tutorial!

HOW TO USE THIS TUTORIAL:
1. Open this file in Emacs with evil-mode and evm loaded
2. Read each lesson — it explains a feature and what keys to press
3. Practice on the text between the ---- lines
4. Press Esc at any time to exit evm and reset
5. Undo with "u" in normal mode if you accidentally change something


Lesson 1: Find Word (C-n)
--------------------------

C-n on a word selects it and enters evm. The word gets highlighted
and all other occurrences in the buffer are shown as dim previews.
Press C-n again to add the next occurrence as a new cursor.

> Try it: place your cursor on the word "apple" below and press C-n.
  You should see "apple" highlighted and other apples shown as previews.
  Press C-n two more times to add cursors on the next two.
  Press Esc when done.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  I like apple pie. She bought an apple at the market.
  An apple a day keeps the doctor away.
  The apple tree in the garden is old. This apple is green.
  We need apple juice and apple cider for the party.
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 2: Find Word from Visual Selection (C-n)
------------------------------------------------

You can also select arbitrary text in visual mode first, then press
C-n. This is useful when you want to match part of a word.

> Try it: enter visual mode (v), select the three letters "get"
  and press C-n. Notice it matches "get" inside longer words too.
  Press C-n a few times to add more. Press Esc when done.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  budget = get_total()
  target = widget.get_value()
  gadget_list = get_gadgets()
  forget(budget_target)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 3: Add Next Match (C-n repeatedly)
------------------------------------------

After the first C-n, each additional C-n adds one more cursor
at the next occurrence. The new cursor becomes the "leader"
(shown in a different color). All cursors move and edit together.

> Try it: place cursor on "color" and press C-n five times.
  Watch how each press adds a new cursor. You should end up with 6.
  Press Esc when done.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  color = "red"
  print(color)
  color = "green"
  print(color)
  color = "blue"
  print(color)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 4: Add Next with n / Navigate Back with N
--------------------------------------------------

Once evm is active, n does the same as C-n — adds the next
occurrence. N does the reverse: adds/jumps to the previous one.
Use n/N when you want to go in a specific direction.

> Try it: place cursor on "data" and press C-n once.
  Then press n three times to jump forward.
  Then press N twice to jump backward.
  Press Esc when done.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  data = load("input.csv")
  print(data)
  data = transform(data)
  save(data)
  data = reload(data)
  validate(data)
  print(data)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 5: Skip Current (q)
----------------------------

Sometimes a match is not what you want. Press q to remove the
current leader cursor and jump to the next occurrence. This lets
you cherry-pick which matches to keep.

> Try it: place cursor on "port" on line 1 and press C-n.
  You want cursors on lines 1, 3, and 5 but NOT on lines 2 and 4.
  Press C-n (or n) to add the next match, q to skip it and jump ahead.
  Sequence: C-n, q, C-n, q, C-n. You should end up with 3 cursors.
  Press Esc when done.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  port = 8080
  port = 3000
  port = 8080
  port = 5432
  port = 8080
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 6: Remove Cursor (Q)
-----------------------------

Q removes the current leader cursor without adding a new one.
The leader moves to the previous cursor (or the next if you
were on the first one).

> Try it: place cursor on "line" and press C-n five times
  to create 5 cursors. Then press Q twice to remove two.
  You should have 3 left. Press Esc when done.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  line 1: first line of text
  line 2: second line of text
  line 3: third line of text
  line 4: fourth line of text
  line 5: fifth line of text
  line 6: sixth line of text
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 7: Select All (\ A)
----------------------------

When you want cursors on every occurrence at once, use \ A
(backslash then Shift-a). This is much faster than pressing
C-n many times.

> Try it: place cursor on "id" and press C-n to start.
  Then press \ A to select all remaining matches at once.
  You should have 8 cursors. Press Esc when done.

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  id: 1001
  id: 1002
  id: 1003
  id: 1004
  id: 1005
  id: 1006
  id: 1007
  id: 1008
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Lesson 8: Putting It Together
-------------------------------

Now combine what you learned. A real editing scenario:
rename the variable "cnt" to "count" in the code below.

> Try it:
  1. Place cursor on "nval" in the text below and press C-n
  2. Press \ A to select all occurrences
  3. You are in extend mode (selections highlighted)
  4. Press c — all selections are deleted, insert mode starts
  5. Type "total" — it appears at all cursor positions
  6. Press Esc to return to normal mode, then Esc again to exit evm

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  nval = 0
  for item in items:
      nval += 1
      if nval > limit:
          break
  print("result:", nval)
  reset(nval)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
