EVM Tutorial 06: Insert Mode
==============================

In cursor mode, press i/a/I/A/o/O to enter insert mode.
What you type appears at ALL cursors simultaneously.
Press Esc to leave insert mode, then Esc to exit evm.

Insert only works from cursor mode (not extend mode).


Lesson 1: Insert (i)
----------------------

i enters insert mode BEFORE each cursor.

> Try it: add a type annotation.
  1. Place cursor on "x" in "x_pos" below, press C-Down twice
  2. Press i — insert mode at all 3 cursors
  3. Type "int " — it appears before each variable name
  4. Press Esc, then Esc

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  x_pos = 0
  y_pos = 0
  z_pos = 0
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
  int x_pos = 0
  int y_pos = 0
  int z_pos = 0


Lesson 2: Append (a)
----------------------

a enters insert mode AFTER each cursor.

> Try it: add a unit suffix.
  1. Place cursor on "0" (the zero) on line 1 below,
     press C-Down twice
  2. Press a — insert mode after each "0"
  3. Type "px" — it appears after each value
  4. Press Esc, then Esc

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  width:  0
  height: 0
  depth:  0
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
  width:  0px
  height: 0px
  depth:  0px


Lesson 3: Insert at Line Start (I)
------------------------------------

I moves all cursors to the first non-blank character,
then enters insert mode.

> Try it: add a keyword.
  1. Place cursor anywhere on line 1 below,
     press C-Down twice
  2. Press I — cursors jump to start of text
  3. Type "export " — it appears at the start of each line
  4. Press Esc, then Esc

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    const API = "/api"
    const TTL = 3600
    const MAX = 100
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
    export const API = "/api"
    export const TTL = 3600
    export const MAX = 100


Lesson 4: Append at Line End (A)
----------------------------------

A moves all cursors to the end of the line,
then enters insert mode.

> Try it: add semicolons.
  1. Place cursor anywhere on line 1 below,
     press C-Down twice
  2. Press A — cursors jump to end of each line
  3. Type ";" — semicolons appear at the end
  4. Press Esc, then Esc

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  let x = 10
  let y = 20
  let z = 30
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
  let x = 10;
  let y = 20;
  let z = 30;


Lesson 5: Open Below (o)
--------------------------

o creates a new line below each cursor and enters
insert mode there. Indentation is preserved.

> Try it: add a print statement after each assignment.
  1. Place cursor anywhere on line 1 below,
     press C-Down twice
  2. Press o — new empty lines appear below each
  3. Type "print(done)" — it appears on all new lines
  4. Press Esc, then Esc

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  a = 1
  b = 2
  c = 3
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
  a = 1
  print(done)
  b = 2
  print(done)
  c = 3
  print(done)


Lesson 6: Open Above (O)
--------------------------

O creates a new line above each cursor and enters
insert mode there.

> Try it: add a comment above each function.
  1. Place cursor on "d" of "def" on line 1 below,
     press C-Down twice
  2. Press O — new lines appear above each
  3. Type "# handler" — it appears on all new lines
  4. Press Esc, then Esc

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  def get():
  def put():
  def del():
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
  # handler
  def get():
  # handler
  def put():
  # handler
  def del():


Lesson 7: Backspace in Insert Mode
------------------------------------

Backspace deletes at all cursors, just like typing
inserts at all cursors.

> Try it: fix a typo.
  1. Place cursor on the second "l" of "colllor"
     on line 1 below, press C-Down twice
  2. Press i — insert mode
  3. Press Backspace once — the extra "l" is removed
     from all three lines
  4. Press Esc, then Esc

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  colllor: red
  colllor: green
  colllor: blue
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
  collor: red
  collor: green
  collor: blue


Lesson 8: Navigate + Insert (Combining with Tutorial 05)
----------------------------------------------------------

The real power: navigate to the right spot first,
then enter insert mode.

> Try it: wrap values in quotes.
  1. Place cursor on "r" of "red" on line 1 below,
     press C-Down twice
  2. Press i — insert mode before the value
  3. Type " — a quote appears before each value
     (electric-pair-mode must be disabled)
  4. Press Esc (exit insert mode, stay in evm)
  5. Press e — move to end of word
  6. Press a — append after cursor
  7. Type " — closing quote after each value
  8. Press Esc, then Esc

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  color: red
  color: green
  color: blue
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result:
  color: "red"
  color: "green"
  color: "blue"

  Note: this requires two insert passes — one for
  the opening quote, one for the closing quote.
  Each Esc returns to cursor mode where you can
  move before inserting again.
