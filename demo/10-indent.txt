EVM Tutorial 10: Indent and Outdent
=====================================

The > and < operators shift lines right or left by one
tab-width (usually 2 or 4 spaces). Like d/c/y, they wait
for a motion — then act at ALL cursors simultaneously.

  >> = indent current line
  << = outdent current line
  >j = indent current + next line
  <k = outdent current + previous line
  >ip = indent inner paragraph
  <ip = outdent inner paragraph

These work only in cursor mode.


Lesson 1: Indent Current Line (>>)
-------------------------------------

>> indents the current line by one tab-width. The cursor
moves to the first non-blank character after indenting.

> Try it: indent the assignment lines.
  1. Place cursor on "x" on line 1 below
  2. Press C-Down twice (3 cursors)
  3. Press >> — all three lines shift right
  4. Press Esc when done (u to undo)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  x = 1
  y = 2
  z = 3
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result (with tab-width 4):
      x = 1
      y = 2
      z = 3


Lesson 2: Outdent Current Line (<<)
--------------------------------------

<< removes one level of indentation. If a line has less
indentation than tab-width, it moves to column 0.

> Try it: outdent the over-indented lines.
  1. Place cursor on "a" on line 1 below
  2. Press C-Down twice (3 cursors)
  3. Press << — all three lines shift left
  4. Press Esc when done (u to undo)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
      a = 1
      b = 2
      c = 3
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result (with tab-width 4):
  a = 1
  b = 2
  c = 3


Lesson 3: Indent with Motions (>j, <k)
-----------------------------------------

>j indents the current line AND the next one (j = down).
<k outdents the current line and the one above (k = up).
This is the same pattern as dj (delete 2 lines) — the
operator + motion covers a range of lines.

> Try it: indent the function bodies.
  1. Place cursor on "print" on line 1 below, press C-n
  2. Press n to add the second "print" (2 matches)
  3. Press Tab to switch to cursor mode
  4. Press >j — each cursor indents its line + the next
  5. Press Esc when done (u to undo)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  def foo():
  print("hello")
  return 1

  def bar():
  print("world")
  return 2
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result (with tab-width 4):
  def foo():
      print("hello")
      return 1

  def bar():
      print("world")
      return 2


Lesson 4: Indent with Text Objects (>ip)
-------------------------------------------

>ip indents all lines in the inner paragraph (lines until
the next blank line). This covers an entire block at once
without needing to count lines.

> Try it: indent the two code blocks separately.
  1. Place cursor on "x" on line 1 below, press C-n
  2. Press n to add the second "x" (2 matches)
  3. Press Tab to switch to cursor mode
  4. Press >ip — each cursor indents its paragraph
  5. Press Esc when done (u to undo)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  x = 1
  y = 2
  z = 3

  x = 10
  y = 20
  z = 30
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result (with tab-width 4):
      x = 1
      y = 2
      z = 3

      x = 10
      y = 20
      z = 30

  <ip works the same way but removes indentation.


Lesson 5: Practical Workflow
-------------------------------

A common task: fix indentation of misaligned code blocks.

> Try it: indent the branches of each if-statement.
  1. Place cursor on first "print" below, press C-n
  2. Press n twice (3 matches of "print")
  3. Press Tab to switch to cursor mode
  4. Press >> — indent just the print lines
  5. Press Esc when done (u to undo)

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  if check_a():
  print("a")

  if check_b():
  print("b")

  if check_c():
  print("c")
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Expected result (with tab-width 4):
  if check_a():
      print("a")

  if check_b():
      print("b")

  if check_c():
      print("c")

  Quick reference:
    >> = indent line              << = outdent line
    >j = indent 2 lines          <k = outdent 2 lines
    >ip = indent paragraph        <ip = outdent paragraph
