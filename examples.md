# EVM Testing Guide

## Quick Start

1. Открыть любой буфер с текстом
2. Поставить курсор на слово
3. Нажать `C-n` — начнётся evm, слово выделится
4. В mode-line появится `EVM[E 1/1]` (Extend mode, 1 курсор)

**Альтернативные способы создания курсоров:**
- `C-Down` / `C-Up` — добавить курсор строкой ниже/выше
- `M-click` — добавить курсор кликом мыши
- `v` + выделение + `C-n` — начать с ограничением на выделенную область

## Test Scenarios

### Scenario 1: Basic Cursor Creation (C-n)

```
Буфер: foo bar foo baz foo
       ^
       курсор здесь
```

1. `C-n` — создаёт первый курсор на "foo", показывает превью всех foo
2. `C-n` или `n` — добавляет курсор на следующий "foo"
3. `C-n` или `n` — добавляет курсор на третий "foo"

**Ожидаемый результат:** 3 курсора, mode-line показывает `EVM[C 3/3]`

### Scenario 2: Vertical Cursors (C-Down/C-Up)

```
Буфер:
line one
line two
line three
^
курсор в начале
```

1. `C-Down` — добавляет курсор на строку ниже
2. `C-Down` — добавляет курсор на третью строку
3. `C-Up` — добавляет курсор выше (wrap around к первой)

**Ожидаемый результат:** 3 курсора в одной колонке

### Scenario 3: Navigation (n/N and ]/[)

После создания нескольких курсоров:

| Клавиша | Действие |
|---------|----------|
| `n` | Перейти к следующему вхождению (создать если нет) |
| `N` | Перейти к предыдущему вхождению (создать если нет) |
| `]` | Перейти к следующему существующему курсору |
| `[` | Перейти к предыдущему существующему курсору |

**Разница n/N vs ]/[:**
- `n/N` — ищут по паттерну, могут создавать новые курсоры
- `]/[` — только перемещают лидера между существующими курсорами

### Scenario 4: Skip and Remove (q/Q)

```
Буфер: foo bar foo baz foo qux foo
```

1. `C-n` на первом foo
2. `q` — пропустить первый foo, перейти ко второму
3. `q` — пропустить второй, перейти к третьему
4. `Q` — удалить текущий курсор (не искать новый)

**Разница q vs Q:**
- `q` — удаляет текущий И находит следующее вхождение
- `Q` — только удаляет текущий курсор

### Scenario 5: Movement (h/j/k/l/w/b/e)

С несколькими курсорами:

1. `l` — все курсоры вправо
2. `w` — все курсоры к следующему слову
3. `b` — все курсоры к предыдущему слову
4. `j` — все курсоры вниз (если многострочный буфер)
5. `0` — все курсоры в начало строки
6. `$` — все курсоры в конец строки

### Scenario 6: Mode Switching (Tab)

```
Буфер: foo bar foo
```

1. `C-n`, `n` — создать 2 курсора на foo
2. `Tab` — переключиться в Extend mode
   - Mode-line: `EVM[E 2/2]`
   - Курсоры превращаются в выделения (по 1 символу)
3. `l`, `l` — расширить выделения на 2 символа
   - Теперь выделено "foo" в обоих местах
4. `Tab` — вернуться в Cursor mode
   - Выделения схлопываются в точки

### Scenario 7: Cursor Mode Editing

В Cursor mode:

| Клавиша | Действие | Тест |
|---------|----------|------|
| `x` | Удалить символ | "foo" → "oo" на всех курсорах |
| `r<char>` | Заменить символ | "foo" → "Xoo" после `rX` |
| `~` | Toggle case | "foo" → "Foo" |
| `i` | Insert mode | Ввод текста на всех позициях |
| `a` | Append | Insert после курсора |
| `o` | Open below | Новая строка под каждым курсором |

### Scenario 8: Extend Mode Editing

1. Создать курсоры, `Tab` для extend mode
2. Расширить выделения (`l`, `w`, `e`)

| Клавиша | Действие |
|---------|----------|
| `y` | Yank (копировать) все регионы |
| `d` | Delete все регионы |
| `c` | Change (delete + insert) |
| `U` | UPPERCASE |
| `u` | lowercase |
| `~` | Toggle case |
| `o` | Flip direction (поменять anchor/cursor) |

### Scenario 9: Yank/Delete/Paste Cycle

```
Буфер: AAA BBB AAA
```

1. `C-n` на AAA, `n` — 2 курсора на AAA
2. `Tab` — extend mode
3. `l`, `l` — выделить "AAA"
4. `y` — yank (скопировать)
5. `d` — delete (удалить)
   - Буфер: " BBB "
   - Автопереход в cursor mode
6. `Tab`, `l`, `l`, `l` — выделить "BBB"
7. `p` — paste (заменить на AAA)
   - Буфер: " AAA "

### Scenario 10: Select All (\A)

```
Буфер: foo bar foo baz foo qux foo
```

1. `C-n` на foo — 1 курсор
2. `\A` (backslash, затем Shift+A) — выбрать все foo

**Ожидаемый результат:** 4 курсора на всех "foo"

### Scenario 11: Align (\a)

```
Буфер:
short = 1
very_long_name = 2
medium = 3
```

1. Поставить курсоры на `=` в каждой строке (через C-Down)
2. `\a` — выровнять

**Ожидаемый результат:**
```
short          = 1
very_long_name = 2
medium         = 3
```

### Scenario 12: Mouse Click (M-click)

Добавление курсоров кликом мыши:

1. `M-click` (Alt+клик) в любом месте буфера — создаёт курсор
2. `M-click` на существующий курсор — перемещает лидера туда

```
Буфер: foo bar baz qux
       ^   ^   ^
       1   2   3  — три M-click создали три курсора
```

**Комбинация с другими способами:**
- `C-n` для первого курсора на слове
- `M-click` для добавления курсоров в произвольных местах

### Scenario 13: Restrict to Region (Visual + C-n)

Ограничение поиска выделенной областью:

```
Буфер: foo bar foo baz foo qux foo
       ^^^     ^^^     ^^^     ^^^
       1       2       3       4   — всего 4 "foo"
```

**Без restriction:**
1. `C-n` на foo, `\A` — выберет все 4 foo

**С restriction:**
1. `v` — войти в visual mode
2. Выделить область от "bar" до "baz" (позиции 5-15)
3. `C-n` на foo внутри выделения — evm активируется с restriction
4. Mode-line покажет `EVM[E 1/1 R]` — буква **R** означает restriction
5. `\A` — выберет только foo внутри restriction (позиции 9)

```
Буфер: foo bar foo baz foo qux foo
           [=========]
           restriction область
               ^
               только этот foo будет выбран
```

**Команды:**
| Клавиша | Действие |
|---------|----------|
| `v` + выделение + `C-n` | Начать evm с restriction |
| `\r` | Снять restriction (искать везде) |
| `n/N` | Поиск только внутри restriction |
| `\A` | Select all только внутри restriction |

**Пример workflow:**
```
Буфер:
function foo() {
  const foo = 1;
  return foo;
}
const foo = "outside";
```

1. Выделить тело функции (visual mode, от `{` до `}`)
2. `C-n` на `foo` внутри выделения
3. `\A` — выберет 3 foo внутри функции, игнорируя foo снаружи
4. `c` — заменить все на новое имя
5. Ввести `bar`, `Esc`

**Результат:**
```
function foo() {
  const bar = 1;
  return bar;
}
const foo = "outside";  // не изменился!
```

### Scenario 14: Exit (Esc)

В любой момент:
- `Esc` — выйти из evm, удалить все курсоры
- Emacs возвращается в обычный режим

## Visual Indicators

- **Обычный курсор:** синий фон
- **Лидер (текущий):** оранжевый фон
- **Регион (extend mode):** зелёный фон
- **Регион лидера:** жёлтый фон
- **Restriction область:** тёмно-серый фон (subtle)
- **Mode-line:** `EVM[C 2/3]` = Cursor mode, лидер #2 из 3
- **Mode-line с restriction:** `EVM[E 1/2 R]` = Extend mode, restriction активен

## Common Issues

### Клавиши не работают
- Проверить: `M-: evm-mode` — должно быть `t`
- Проверить: `M-: (key-binding "n")` — должно быть `evm-find-next`

### Курсоры не видны
- Проверить тему — возможно цвета совпадают с фоном
- `M-x customize-face RET evm-cursor-face`

### n/N не добавляют курсоры
- Все вхождения уже заняты — n/N просто перемещают лидера
- Используйте `\A` для выбора всех сразу
