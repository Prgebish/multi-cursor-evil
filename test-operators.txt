--------------------------------

  alpha beta gamma delta

Test visual-line (\ c):
  line one
  line two
  line three
  - V, jj (select 3 lines)
  - \ c -> 3 cursors at line starts

Test visual-block (\ c):
  col1 col2
  col1 col2
  col1 col2
  - C-v, jj (block select)
  - \ c -> 3 cursors at block column


9.2 Multiline Mode (M)
----------------------
word word word
  - C-n on "word", M -> toggles multiline search
  - Mode line shows status


9.3 Undo/Redo (u / C-r)
-----------------------
foo bar foo baz foo
  - C-n, n, n (3 cursors on "foo")
  - c, type "XXX", Esc
  - u -> undo, cursors stay synced to "foo"
  - C-r -> redo


9.4 Reselect Last (\ g S)
-------------------------
hello world hello
  - C-n, n (2 cursors on "hello")
  - Esc (exit evm)
  - \ g S -> cursors restored with mode

Phase 10: evil-surround Integration
====================================

10.1 Surround in Extend Mode (S)
--------------------------------
foo bar foo baz
  - C-n on "foo", n (2 cursors on "foo")
  - S " -> обе "foo" окружены кавычками
  - Result: "foo" bar "foo" baz

hello world hello
  - C-n, n, S ) -> окружить скобками
  - Result: (hello) world (hello)


10.2 Surround with Motion (ys)
------------------------------
hello world test
  - C-n on "hello", n on "test" (2 cursors)
  - Tab (cursor mode)
  - y s i w " -> окружить inner word кавычками
  - Result: "hello" world "test"

foo bar
baz qux
  - C-down (2 cursors на foo и baz)
  - y s $ ) -> окружить до конца строки скобками
  - Result: (foo bar)
           (baz qux)

line one
line two
  - 2 cursors
  - y s s " -> окружить всю строку (yss)
  - Result: "line one"
           "line two"


10.3 Delete Surround (ds)
-------------------------
"hello" world "test"
  - Cursors inside both quoted words
  - d s " -> удалить кавычки
  - Result: hello world test

(foo) and (bar)
  - Cursors inside parens
  - d s ( -> удалить скобки
  - Result: foo and bar


10.4 Change Surround (cs)
-------------------------
"hello" world "test"
  - Cursors inside quotes
  - c s " ' -> заменить " на '
  - Result: 'hello' world 'test'

(foo) and (bar)
  - Cursors inside parens
  - c s ( [ -> заменить () на []
  - Result: [foo] and [bar]

<div>text</div>
  - Cursor inside tag
  - c s t " -> заменить tag на кавычки
  - Result: "text"


Quick Reference
===============
\ c     - Create cursors from visual selection
M       - Toggle multiline mode
u       - Undo with cursor sync
C-r     - Redo with cursor sync
\ g S   - Reselect last cursors
" a y   - Yank to register 'a'
" a p   - Paste from register 'a'
" A y   - Append to register 'a'

evil-surround:
S char  - Surround regions (extend mode)
ys motion char - Surround motion (cursor mode)
ds char - Delete surround
cs old new - Change surround
